# Siggy Bot Pro Full Dashboard

Discord bot with a live web dashboard.

## Setup

1. Install Node.js
2. Run:

npm install

3. Create `.env` file

TOKEN=YOUR_DISCORD_BOT_TOKEN

4. Start bot

npm start

## Deploy to Railway

1. Upload to GitHub
2. Create Railway project
3. Deploy repo
4. Add env variable:

TOKEN=your_bot_token

Then open:

https://your-project.up.railway.app
